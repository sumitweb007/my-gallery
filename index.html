Setup Firebase 🔥

Paste your Firebase config to enable Google login, cloud photo storage, and sync across all your devices.

// Firebase Console → Project Settings → Your Apps → Web const firebaseConfig = { apiKey: "AIza...", authDomain: "your-app.firebaseapp.com", projectId: "your-app", storageBucket: "your-app.appspot.com", messagingSenderId: "123456", appId: "1:123456:web:abc" };
5-minute setup:
1 · Go to console.firebase.google.com → New project
2 · Add a Web App → copy the config object
3 · Enable Authentication → Email/Password + Google
4 · Enable Firestore Database → test mode
5 · Enable Storage → test mode
6 · Paste config above and hit Launch!
